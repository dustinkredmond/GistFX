Welcome!


Read this as this is the ONLY time you will see this message!


You will need a GitHub personal access token before you can use this program. Click on the question mark (next screen) for detailed instructions explaining how to create one.


Once you have a valid access token, paste it into the Token field, then type in a password that you will use to log into GistFX. Your password is used to encrypt then store your personal access token locally for your convenience.


GistFX NEVER stores clear text versions of your password or your token and every measure has been taken to ensure your privacy, because your privacy matters to us!


See Help in the app for more information.


Enjoy the program!
